🔬 Research
CLOUD MORPH – Hybrid Optimization for Cloud Workload Migration

Published at International Conference on Data Science and Applications (Springer Proceedings)

Proposed a hybrid optimization framework combining:

Improved Dynamic Multi-Objective Optimization Algorithm (IDMOA)

Bidirectional Residual Recurrent Neural Network (BResRNN)

The system enables trust-aware workload migration in heterogeneous cloud environments, achieving:

lower operational cost

improved energy efficiency

faster task response time

compared with classical optimization methods such as Genetic Algorithms (GA) and Particle Swarm Optimization (PSO).

CIGIN – Graph Neural Networks for Molecular Interaction Prediction

Reproduced and extended the Chemically Interpretable Graph Interaction Network using:

PyTorch

RDKit

Graph Neural Networks

The model predicts solvation free energy while improving interpretability through attention pooling, contributing to applications in computational chemistry and drug discovery.
